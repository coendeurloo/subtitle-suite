# Subtitle Suite - Beta 0.9.2

Release date: 2026-03-29

## Highlights

- Manual Smart Sync now applies synced subtitles directly to active playback (single or dual).
- Fixed an issue where playback could remain on a single subtitle (for example English reference) after manual Smart Sync.
- Smart Sync subtitle pickers now filter by the current episode in season folders.
- Renamed menu action to `Smart Sync Subtitles (Manual)...`.
- Manual Smart Sync now closes subtitle/menu overlays and ends with one clear success/failure result popup.

## Files

- Addon version bumped to `2.9.4` for Kodi update/install flow.
